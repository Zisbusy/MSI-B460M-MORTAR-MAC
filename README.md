# MSI B460M MORTAR 黑苹果 引导文件

 - MSI B460M MORTAR
 - i5-10500
 - Fenvi FV-T919 BCM94360CD

## 安装
本引导高度依据个人喜好修改，默认无oc启动界面。请自行添加三码。     
适用系统：Big Sur，Monterey。
